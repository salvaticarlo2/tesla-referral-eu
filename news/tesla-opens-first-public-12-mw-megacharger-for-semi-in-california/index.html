Tesla opens first public 1.2 MW Megacharger for Semi in California

Jul 22, 2026 · Source: electrive

Tesla has opened its first full-scale public Megacharger station for the Semi in California. The new site in Bloomington features six charging stalls delivering up to 1.2 MW each and supports the latest-generation Tesla Semi with the MC2 charging connector.
