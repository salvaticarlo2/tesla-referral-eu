Cómo usar el código referido Tesla en España

Tres pasos simples. El enlace de referido debe usarse antes de configurar tu vehículo — no podrás añadirlo después del pedido.

01

Abrir el enlace de referido

Accede a tesla.com/referral/carlo719460. Esto abre directamente el configurador Tesla español con el código referido ya aplicado. No empieces desde la página de inicio de tesla.com/es_es — el código no se aplicará.

02

Configurar tu Tesla

Elige tu modelo (Model 3, Model Y, Model S o Model X), color, llantas e interior. El beneficio del referido aparece en el resumen del pedido — Supercarga gratuita o créditos Tesla según el modelo. Si no tienes cuenta, crea una en tesla.com con tu dirección de correo española.

03

Realizar el pedido

Finaliza tu pedido normalmente. Los beneficios solo se aplican con un referido activo y se acreditan tras la entrega.
