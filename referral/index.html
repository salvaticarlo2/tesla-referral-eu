Frequently Asked Questions

What do I get with a Tesla referral code in Europe?

Using a referral code gives you savings on a new Tesla in Europe. For Model 3 and Model Y, you typically receive free Supercharging kilometres. For Model S and Model X, you receive Tesla credits. In many European countries there is also a direct monetary discount of approximately €500 applied at order.

Can I apply a referral code after placing my order?

No. The referral must be active before you place your order. Once submitted, Tesla does not allow referral codes to be added retroactively. Always start from the referral link before configuring your vehicle.

Does the referral work if I order from a different country than where I live?

Generally no. Tesla's configurator is tied to the country/region from which you order. Use the referral link and select your correct country within the Tesla website — the code will apply as long as it's active in your country.

How long do the Supercharging credits last?

Free Supercharging kilometres and Tesla credits earned through referral typically expire 12 months after being awarded (usually at delivery). Use them within that window.
